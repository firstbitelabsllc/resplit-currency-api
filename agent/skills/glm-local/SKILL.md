# GLM Local

GLM local work is optional and gated for this repository.

Allowed:

- Preserve handoff context about GLM readiness.
- Record whether a local model server is absent or externally supplied.
- Keep project instructions compatible with a future local model picker.

Not allowed:

- Download model weights.
- Start paid or hosted model calls.
- Assume a GLM runtime exists without a local receipt.
